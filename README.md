# Picpic

## 简介

水群的时候无聊写的东西，可以把一张图片用自己拼起来，效果如图：

![preview](https://github.com/will7101/PicPic/blob/master/preview.jpg)

**图形界面等待填坑。**

## 使用方法

在文件目录下打开命令行，输入
```shell
python3 picpic.py [-h] [-x width] [-y height] [-a alpha] <filename>
```
即可。

- -x -y 选项可以指定生成图片的“像素”个数
- -a 选项可以指定生成图片与原图的透明度（混合比例）
